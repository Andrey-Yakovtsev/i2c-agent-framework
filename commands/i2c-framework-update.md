Обновить I2C фреймворк до последней версии.

Выполняет git pull в папке фреймворка, перегенерирует установленные файлы в текущем проекте
и показывает changelog последних изменений.

**Аргументы:** не требуются.
**Пример:** `/i2c-framework-update`

Выполни следующие шаги:
1. Запусти `cd {{FRAMEWORK_DIR}} && git pull` (используй Bash tool)
2. Запусти `{{FRAMEWORK_DIR}}/install.sh` с аргументом — путь к текущему проекту (используй Bash tool)
3. Покажи пользователю changelog: `cd {{FRAMEWORK_DIR}} && git log --oneline -10`
