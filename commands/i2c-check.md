Проверить консистентность всей документации проекта.

Запускает диагностический агент, который сверяет PRD, ADR и RFC между собой:
находит конфликты, недостающие зависимости, несоответствия. Пишет отчёт в `.i2c/scratch/consistency-report.md`.

**Аргументы:** не требуются.
**Пример:** `/i2c-check`

Прочитай файл {{FRAMEWORK_DIR}}/orchestrator-installed.md (используй Read tool) и выполни команду `check` по инструкциям из этого файла.
