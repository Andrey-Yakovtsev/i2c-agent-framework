Создать Product Requirements Document через полный конвейер агентов.

Запускает: Supervisor (pre-flight) → Researcher → Architect → Critic → Writer → Supervisor (post-review).
Результат сохраняется в `docs/PRD.md` после одобрения Supervisor.
Использовать: начало нового проекта, значительный pivot, расширение на новый product area.

**Аргументы:** не требуются.
**Пример:** `/i2c-create-prd`

Прочитай файл {{FRAMEWORK_DIR}}/orchestrator-installed.md (используй Read tool) и выполни команду `create-prd` по инструкциям из этого файла.
