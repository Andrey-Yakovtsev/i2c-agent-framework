Инициализировать проект для работы с I2C фреймворком.

Создаёт структуру `.i2c/` (config, MEMORY, GOALS, JOURNAL, pipeline_state) и подключает фреймворк через CLAUDE.md.
Для нового проекта — интерактивно заполняет config.md.
Для существующего — запускает Researcher-агента для аудита кодовой базы и заполнения MEMORY.md.

**Аргумент:** путь к проекту (опционально, по умолчанию — текущая директория).
**Пример:** `/i2c-setup` или `/i2c-setup /path/to/project`

Выполни команду I2C `setup $ARGUMENTS` из инструкций оркестратора.
